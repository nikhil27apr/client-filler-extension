# Client Form Filler Extension

This extension adds a toolbar button for Chrome or Edge that can fill the Add Client form using values from the popup.

## Install in Chrome or Edge

1. Open Chrome or Edge.
2. Go to `chrome://extensions` or `edge://extensions`.
3. Turn on "Developer mode".
4. Click "Load unpacked".
5. Select this folder: `client-filler-extension`.
6. Open the Add Client page and click the extension icon.

## Usage

- Prefix: type anything, or use `PCA` / `NV`.
- Index: leave blank for no index; enter a number to append it to names, email, and phone.
- Phone and Email: leave blank to auto-generate.

The extension will fill the fields, expand Additional Fields, and scroll to the Save button.
